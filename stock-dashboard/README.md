# AI Stock Market Intelligence Dashboard

## Project Overview
This project is an AI-powered stock market dashboard built using Python and Streamlit. It displays real-time stock information, stock price charts, and AI-based stock predictions.

## Features
- Real-time stock data
- Stock price visualization
- Current, highest, and lowest stock prices
- AI-based next-day stock prediction
- Buy/Sell recommendation
- Interactive dashboard

## Technologies Used
- Python
- Streamlit
- Yahoo Finance (yfinance)
- Pandas
- Scikit-learn
- Joblib

## How to Run

Install required packages:

pip install -r requirements.txt

Run the application:

streamlit run app.py
